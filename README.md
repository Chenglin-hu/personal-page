# 🌲 程檩的个人主页

一个现代化、响应式的个人主页，使用原生 HTML/CSS/JavaScript 构建。

## ✨ 功能特性

### 核心功能
- 🌓 **深色模式** - 一键切换亮/暗主题，自动保存偏好
- 🎨 **精美设计** - 渐变背景、卡片阴影、流畅动画
- 📱 **响应式布局** - 完美适配手机、平板、桌面
- ⚡ **流畅交互** - 3D 卡片倾斜、悬停效果、过渡动画

### 互动功能
- 💬 **留言板** - 访客可以留言互动，支持本地存储
- 🎵 **音乐播放器** - 内置音乐播放功能
- ❄️ **雪花效果** - 动态雪花飘落背景
- 🎯 **技能标签** - 点击显示详细说明泡泡

### 个性化元素
- 📝 **签名轮播** - 自动切换个性签名
- 📊 **访问计数** - 显示页面访问次数
- 🏆 **项目展示** - 展示个人项目经历
- 📈 **技能进度** - 可视化展示技术栈水平

## 🚀 快速开始

### 方法 1：直接打开
```bash
# 直接用浏览器打开 index.html
open index.html
```

### 方法 2：本地服务器
```bash
# 使用 Python 启动本地服务器
python3 -m http.server 8080

# 访问 http://localhost:8080
```

### 方法 3：使用 VS Code Live Server
1. 安装 Live Server 扩展
2. 右键 index.html → "Open with Live Server"

## 🎨 自定义指南

### 修改个人信息
编辑 `index.html` 中的以下内容：
- 姓名、头衔、简介
- 社交链接（邮箱、GitHub 等）
- 技能标签
- 项目经历

### 修改配色方案
编辑 `style.css` 中的 CSS 变量：
```css
:root {
    --primary-color: #569688;      /* 主色调 */
    --accent-color: #f39c12;       /* 强调色 */
    --bg-start: #f5f7fa;           /* 背景渐变起始色 */
    --bg-end: #c3cfe2;             /* 背景渐变结束色 */
}
```

### 修改个性签名
编辑 `script.js` 中的 `signatures` 数组：
```javascript
const signatures = [
    '『根深者，必成参天』',
    '『你的签名』',
    '『更多签名』
];
```

### 更换头像
1. 准备一张正方形图片（建议 300x300 以上）
2. 命名为 `pic_20260110212306_0.jpg`
3. 放到项目根目录
4. 或修改 `index.html` 中的 `src` 属性

### 修改音乐
编辑 `script.js` 中的音频 URL：
```javascript
audio = new Audio('你的音乐 URL.mp3');
```

## 🌐 部署上线

1. 拖拽项目文件夹到 Netlify
2. 自动部署完成
